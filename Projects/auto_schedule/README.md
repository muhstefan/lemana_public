# AutoSchedule 0.97

Программа для автоматического переноса расписаний из Google Sheets во внутреннюю CRM системы компании.

##  Возможности
-  Получение расписаний из Google Sheets
-  Автоматический перенос расписания в CRM
-  Гибкая настройка через различные режимы работы
-  Подробное логирование всех операций

##  Установка
Подробные инструкции по установке доступны в файле `MAIN_README.md`

### Основные команды

#### Полная очистка/создание расписания
- `очистка` - полностью очищает расписание за указанный месяц
- `создание` - полностью заполняет расписание за указанный месяц

#### Работа с конкретным сотрудником
- `очистка [ЛДАП] [дата1] [дата2]`  
  Пример: `очистка 6018582 24 26`  
  Очищает расписание указанного сотрудника в заданном интервале дат

- `создание [ЛДАП] [дата1] [дата2]`  
  Пример: `создание 6018582 24 26`  
  Заполняет расписание указанного сотрудника в заданном интервале дат

#### Работа со всеми сотрудниками
- `очистка [дата1] [дата2]`  
  Пример: `очистка 24 26`  
  Очищает расписание всех сотрудников в заданном интервале

- `создание [дата1] [дата2]`  
  Пример: `создание 24 26`  
  Заполняет расписание всех сотрудников в заданном интервале

### Важно
Название исполняемого файла (EXE) через дефис определяет направление (мир), к которому будет применена программа
